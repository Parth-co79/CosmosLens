## 📖 About

CosmosLens is a dark-themed web gallery built using NASA's APOD API. Users can explore 
astronomy pictures from any date in history — including their own birthday.

---

## 🛰️ API Used

- **Name:** NASA APOD API  
- **Endpoint:** `https://api.nasa.gov/planetary/apod`
- **Auth:** Free API Key from [api.nasa.gov](https://api.nasa.gov/)

---

## ✨ Features

- 📅 Date Picker — fetch any historical APOD
- 🎂 Birthday Universe — see the cosmos on your birthday
- 🖼️ Full-Screen Modal with HD Toggle
- 🌑 Dark space theme with glowing effects
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

`HTML5` `CSS3` `JavaScript`

---

## ⚙️ Setup

1. Clone the repo — `git clone (https://github.com/Parth-co79/CosmosLens)`
2. Open `index.js` and add your NASA API key:
```javascript
   const API_KEY = 'YOUR_KEY_HERE';
```
3. Open `index.html` in your browser — no server needed.
